## 目的

このアプリは、暗い場所等での不用意な画面OFF・画面ON がなくなることを目的としています。
また、個人として以下の目的で作成しました。

* Kotlin の勉強
* 透明 startForeground の動作検証

## 機能

* 近接センサーで実施するのは画面の ON のみ
* OFF は常駐エリア、またはハードキーで実施
